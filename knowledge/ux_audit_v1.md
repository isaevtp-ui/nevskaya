# UX-аудит прототипа главной страницы — v1 (2026-03-30)

## Цели сайта
- Портфолио команды
- Визитка
- Привлечение новых игроков
- Вовлечение детей в спорт
- Создание значимости и важности
- Полезные материалы для родителей

## Проблемы

### 1. Нет эмоции
Тёмная тема + строгие карточки = стиль IT-стартапа или киберспорта. Аудитория — родители и дети 9-10 лет, нужно тепло и энергию.

### 2. Hero без фото — мёртвый
Градиент вместо фото команды. Одна живая фотография девочек на площадке важнее всего текста на странице.

### 3. Данные-заглушки
"Анна И.", "Иванов И.И." — убивают доверие. Лучше не показывать секцию, чем с фейковыми данными.

### 4. Структура не под цели
Текущий порядок — просто перечисление блоков без воронки.
Нужна логика: эмоция → доверие → ответы на вопросы → действие.

### 5. Нет ответов на вопросы родителей
- Сколько стоит?
- Что нужно с собой?
- Подойдёт ли новичок?
- Где это?
- FAQ или блок "Для новичков" — must have.

### 6. Форма записи слишком формальная
"Отправить заявку" — звучит казённо. Родители не заполняют формы, они пишут в мессенджер. Нужна ссылка на Telegram/WhatsApp тренера + привязка к ближайшей открытой тренировке.

## Рекомендации

- Светлая тема с синими акцентами — ближе к семейной аудитории
- Большое фото команды в hero
- Блок "Попробуй бесплатно" — открытая тренировка с датой + "Написать тренеру"
- Блок достижений — фото, названия турниров, места
- FAQ для родителей (5-7 вопросов)
- Telegram/WhatsApp вместо формы (или вместе)
- Меньше секций, больше глубины в каждой
- Структура воронки: эмоция → значимость → доверие (тренеры, достижения) → полезность (статьи, FAQ) → действие (записаться)
