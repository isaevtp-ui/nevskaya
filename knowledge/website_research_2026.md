- Jersey Watch — платформа для youth sports, чистый дизайн

### Россия

## Обязательные разделы сайта
1. Главная (герой + ключевая инфо)
2. Расписание / Календарь
3. Состав команды (ростер)
4. Фото/видео галерея
5. Новости и объявления
6. О команде / Тренеры
7. Контакты
8. Запись / Регистрация

## Сравнение платформ
### Tilda — РЕКОМЕНДАЦИЯ для нетехнических пользователей
- Personal: 750 руб/мес (6000 руб/год)
- Business: 1250 руб/мес (12000 руб/год)
- Плюсы: drag-and-drop, русский интерфейс, хостинг включён
- Минусы: нет спортивных плагинов, ограниченный SEO

### WordPress
- Бесплатный CMS + хостинг от ~300 руб/мес
- SportsPress плагин (бесплатный, Pro $99/год)
- Плюсы: полная кастомизация, спортивные плагины
- Минусы: сложнее для нетехнических, нужен хостинг

### Join Sport (joinsport.io)
- Спецплатформа для спортивных сайтов в России
- Турнирные таблицы, live-трансляции
- Подходит для лиг и федераций

## Тренды дизайна 2025-2026
- Яркие насыщенные цвета, высокий контраст
- Mobile-first обязательно
- Крупная типографика
- Микро-анимации
- Role-based навигация (родители/игроки/тренеры)
- Hero-секции с фото/видео
