# Исследование: сайт детской баскетбольной команды

## Платформа: Tilda vs WordPress
- Tilda — проще, быстрее, дешевле в обслуживании, идеально для нетехнических админов
- WordPress + SportsPress — мощнее для статистики и турнирных таблиц, но требует обслуживания
- Рекомендация: Tilda Personal (6000 руб/год) для старта

## Обязательные разделы
1. Главная (hero + ближайшие события)
2. Расписание (тренировки, игры, турниры)
3. Состав команды (фото, номера, позиции)
4. Тренеры (фото, био)
5. Новости (результаты, объявления)
6. Галерея (фото/видео)
7. О команде (история, клуб)
8. Контакты + запись в команду

## Тренды дизайна 2025-2026
- Яркие цвета, высокий контраст
- Крупная типографика
- Mobile-first
- Hero с фото/видео команды
- Микроанимации
